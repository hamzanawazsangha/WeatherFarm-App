# Assets Directory

This directory contains static assets for the application.

## Required PWA Icons

For PWA functionality, you'll need to add the following icon files to the `/public` directory:

- `pwa-192x192.png` - 192x192px icon
- `pwa-512x512.png` - 512x512px icon
- `favicon.ico` - Standard favicon
- `apple-touch-icon.png` - Apple touch icon (180x180px)
- `mask-icon.svg` - Mask icon for Safari

You can generate these icons using tools like:
- [PWA Asset Generator](https://github.com/onderceylan/pwa-asset-generator)
- [RealFaviconGenerator](https://realfavicongenerator.net/)
